## 🚀 快速开始

### 3 步启动
```bash
# 1. 推送代码
git push origin main

# 2. 等待构建（5-10 分钟）
# 访问 https://github.com/YOUR_USERNAME/gomoku-online/actions

# 3. 运行应用
docker pull ghcr.io/YOUR_USERNAME/gomoku-online:main
docker run -d -p 3000:3000 ghcr.io/YOUR_USERNAME/gomoku-online:main
```

### 访问应用
```
http://localhost:3000
```
